# Obsidian-chem
My personal obsidian plugin for working with chemistry and physics.

## Features
- Hotkey for inserting a math environment `$$`
- Hotkey for inserting an mhchem environment `$\ce{}$`
- Leader key system for inserting greek letters:
    - Leader + a = `\alpha`
    - Leader + A = `\Alpha`
    - Leader + z = `\zeta`
